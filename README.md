## Require

  - Newtonsoft.Json

 
 
 # Features
 - Support `Load GameData From GoogleSheet` for DevMode
 - Support `Write GameData To GoogleSheet` for DevMode
 - Support `Local Data Cache` for Release
 - Support `Code Generator `
 - `Google Drive Explorer` GUI
